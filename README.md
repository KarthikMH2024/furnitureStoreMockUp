# Furniture Store Mockup

This project is a mockup of a simple furniture store web application. It includes a basic layout with a top bar, sidebar, and main content area, all designed to mimic the look and feel of a furniture store website.

## Features

- **Top Bar (top.html):** Contains a logo and links to different sections of the website, such as the home page, store page, and feedback page.
- **Sidebar (sidebar.html):** Provides links related to user accounts (sign in, register, my account, orders) and product categories (sofas, chairs, tables, couches). It also includes contact information for the store.
- **Main Content Area (frame1.html and home.html):** Displays the content corresponding to the selected links in the sidebar.

## Pages

- **about.html (About Us Page):** Contains information about the furniture store, its history, mission, and values.
- **catalogue.html (Catalogue Page):** Displays the different categories of furniture products available in the store.
- **feedback.html (Feedback Page):** Provides a form or other means for customers to submit feedback or reviews about their experience with the store.
- **frame1.html (Frame 1):** A frame within the frameset, displaying additional content related to the selected category in the sidebar.
- **frame2.html (Frame 2):** Another frame within the frameset, displaying the main content of the selected page.
- **home.html (Home Page):** The main landing page of the website, providing an overview of the store and its offerings.
- **index.html (Index Page):** The main HTML file that includes the frameset layout and references to the other HTML files.
- **store.html (Store Page):** Provides an overview of the products available in the store and possibly allows customers to browse and purchase items.
- **user_account.html (User Account Page):** Provides options for users to sign in, register, or manage their account information.

## Technologies Used

- HTML
- CSS

## Usage

1. Clone the repository: `git clone https://github.com/your-username/furniture-store-mockup.git`
